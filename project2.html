Dall-E Poster

Dall-E

Project Description: For this project, I was assigned to go on Dall-E, which is a photo generating website, where an OpenAi program generates certain images from language descriptions called, "prompts." So for this image, I wanted sort of like a medieval and steampunk combined theme, so I generated images that included, pirate ships, dragons and medieval castles in the landscape. Also, I wanted this image to look like a painting from the Renaissance era.
